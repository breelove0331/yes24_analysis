# 📊 YES24 IT 베스트셀러 데이터 분석 프로젝트

## 1. 프로젝트 개요
본 프로젝트는 YES24 IT/컴퓨터 카테고리 실시간 베스트셀러 데이터를 수집하고,  
데이터 분석을 통해 도서 판매 트렌드 및 성공 요인을 파악하는 것을 목표로 합니다.

---

## 2. 분석 목적
- 베스트셀러 도서의 공통적인 특징 분석
- 판매지수에 영향을 미치는 주요 요인 탐색
- 도서 기획 및 마케팅 전략 수립을 위한 데이터 기반 인사이트 도출

---

## 3. 데이터 수집
- YES24 웹 페이지 HTML 크롤링
- IT/컴퓨터 카테고리 실시간 베스트셀러 데이터

주요 변수:
- 판매지수
- 리뷰 수
- 평점
- 가격
- 출판사

---

## 4. 분석 내용

### 📌 출판사 점유율 분석
- 특정 출판사가 높은 비중을 차지
- 브랜드 영향력 및 시장 집중 현상 확인

### 📌 가격 분포 분석
- IT 도서는 2~3만원대에 집중
- 시장에서 수용 가능한 가격대 존재

### 📌 리뷰수 vs 판매지수
- 일부 양의 관계 존재
- 하지만 완전한 상관관계는 아님
- 다양한 요인이 복합적으로 작용

### 📌 리뷰 효율성 분석
- 리뷰 효율성 = 리뷰수 / 판매지수
- 고효율 도서: 독자 반응이 높은 도서
- 저효율 도서: 판매 대비 리뷰 참여가 낮은 도서

---

## 5. 주요 인사이트

- IT 베스트셀러는 특정 출판사에 집중되는 경향이 있음
- 가격은 2~3만원대 중심으로 형성됨
- 리뷰수와 판매지수는 완전한 상관관계가 아님
- 리뷰 외에도 브랜드, 트렌드 등의 요인이 판매에 영향 가능
